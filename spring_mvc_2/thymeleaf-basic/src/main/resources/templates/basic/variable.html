SpringEL 표현식

    Object
  • ${user.username} =
  • ${user['username']} =
  • ${user.getUsername()} =
    List
  • ${users[0].username} =
  • ${users[0]['username'] =
  • ${users[0].getUsername()} =
  • List 에서 users[0] 인덱스를 넣으면 user Object 가 나옴
    Map
  • ${userMap['userA'].username} =
  • ${userMap['userA']['username'] =
  • ${userMap['userA'].getUsername()} =
  • Map 에서 userMap['userA'] key 를 넣으면 user Object 가 나옴
